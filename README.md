#  Gym Management System API

A Flask-based REST API for managing a gym system including members, subscriptions, attendance, payments, and class bookings.

---

##  Features

- Member management
- Subscription tracking
- Attendance check-in & check-out
- Payment tracking
- Class booking system
- RESTful API design

---

##  Tech Stack

- Python (Flask)
- Flask-SQLAlchemy
- MySQL
- PyMySQL
